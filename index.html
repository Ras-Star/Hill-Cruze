How To Play

Seamless run flow

Steer: Left / Right or A / D
Jump: Up / W / Space
Duck: Down / S
Boost: Shift or the on-screen pedal button

Each run gives score, distance, and tokens. Distance milestones trigger biome swaps and bonus tokens, while total lifetime tokens unlock new rider looks, frames, badges, and hub background packs.

Boot, preload, menu, run, milestone, pause, and game-over moments all happen in Phaser, while Bootstrap frames the shell, stats, and responsive layout around the canvas.
